# Self-Hosted Outreach With oXRM

## The Problem

Outreach context gets scattered across spreadsheets, inboxes, LinkedIn, notes,
and AI chats.

## The oXRM Loop

Target -> context -> draft -> approval -> external action -> event ->
follow-up.

## Use Cases

- job search
- customer outreach
- partnerships
- investor outreach
- consulting leads

## What Agents Can Do

- read queues
- summarize context
- draft next actions
- update records
- create tasks

## What Humans Control

- sending
- approvals
- real external actions
- final judgment
